宫崎骏电影

看宫崎骏的电影总给人很舒服的感觉, 会让你觉得世界总是美好的, 总可以遇到和蔼亲切的人, 总可以给别人完全的信任, 这是在现实生活中所很难做到的, 我觉得这也宫崎骏的电影在各个年龄段都比较受欢迎的原因吧.

飞越疯人院

一部宣扬对自由的渴望的电影, 不论什么时候都不能陷入一个温和的环境, 不敢走向更广阔的外面的世界, 剧中不止一次提到, 你为什么还在这里, 为什么不出去? 人们总是喜欢温水煮青蛙的环境, 慢慢地适应温暖的环境就慢慢失去了走出去的勇气.

飞屋环游记

一个缺少子女的老人和一个缺少父爱的孩子, 老人只为了守护他的房子, 去寻找天堂瀑布, 孩子只为了要那一枚协助老人的徽章, 执意帮助老人寻找那只鵴鸟. 然而实际上, 老人是想实现妻子少时的梦想, 孩子是想让父亲亲手给他戴上徽章. 这些可能在别人看来很无聊的事情对于当事人而言却是最重要的呀.

寻梦环游记

有人说人的死亡分为三次, 第一次是你肉体的死亡, 第二次是你认识的人都已经忘记了你, 第三次是你不认识的人也已经忘记了你, 这部电影告诉我们家人对于你的记忆是有多么的重要. 一部看似是讲述追逐音乐梦想的孩子的故事, 实际上我觉得是讲述这个世界上你的家人一直想念着你是一件多么幸福的事情. 从最初的爸爸抛下妻儿找寻自己的吉他梦, 到后来太太太奶奶也开始支持这个孩子的音乐梦都在告诉我们家人才是你我生命中最重要的组成部分.

绿皮书

在看这部电影之前知道是将种族歧视的, 影片中的确表现出来美国当时对黑人的区别对待甚至到了原则性的问题. 不过这里我感受更深的在于唐和托尼之间的不一样的友谊. 虽说情谊往往“门当户对”的才会长久, 但是这两个人的出身, 背景等等都是相差甚远, 最终却也结下了深厚的友谊. 一方面来自于唐本身难以处理跟市井街民之间的矛盾, 但是托尼却颇有研究, 另一方面唐本身也是孤独的, 尽管他是一名大音乐家, 但是身为黑肤色的人却鲜有交心的朋友, 连唯一的兄弟也不怎么联系. 从托尼身上看到了其实活着最重要的是让自己开心, 让自己的家人开心才重要. 剧中唐曾情绪爆发过:“因为我不够黑, 也不够白, 甚至不够男人, 那么托尼告诉我, 我到底是什么”, 其实是什么并不重要, 人不是为了他人的评价而活着. 这种友谊很想<<触不可及>>中二人的友谊, 都是建立于上层社会的人和底层社会的人之间的. 我觉得其实从人的本性来看, 都是会倾向于“放松自我”生活方式的, 就像剧中唐虽然最开始抵触手拿着炸鸡吃, 但是最后也还是觉得吃的很开心. 从广义上讲, 咸鱼的确是一件很令人舒服的事情…, 当然这是开个玩笑, 就像<<触不可及>>中戏弄警察之后的笑声, <<泰坦尼克号>>中女主在下等舱跳芭蕾舞后兴奋的表情, 与这里唐在黑人区弹喜爱的古典钢琴曲流露出来的笑容是类似的, 那都是发自内心的开心. BTW, 其实我们做什么有的时候并不重要,重要的是带着什么心情, 跟谁一起做…

死亡诗社

这部电影给我最大的感触在于所谓的“长大了你就明白了”这都是一个幌子而已, 所谓“都是为你好”也只是没有说辞的借口罢了, 这里重要的是你就需要服从父母, 学校, 上级的安排, 因为当你长大的时候, 你或许会迫于现实的压力, 或者也熄灭了年少的激情, 也就真的 “明白了”. 而且人们总是以为自己认为对的事情别人也应该这样觉得, 不然就是错误. 而且当出了事情一定需要替罪羊来顶罪, 因为我属于官方, 我是不会有错的. 这样的嘴脸在<<放牛班的春天>>, <<闻香识女人>>都有出现过, 但是这里更加尖锐, 因为迫于父母压力而自杀的学生需要一个让学生独立思考的老师来承担所有责任, 因为老师一直鼓励学生演话剧, 一直违反学生父母的意愿, 而父母希望他读医, 所以老师以被辞退了. 老师代表的是向往理想的为了自己的梦想而活着的人, 学校和家长代表着你需要成为我希望的成为的人, 毕竟前者是弱势, 毕竟学生也不得不向家长学校妥协, 所以即便有那么一丝光明照向黑暗, 也会被淹没在无尽的黑暗中.

剧中也不止一次出现你想要改变这所学校的原则嘛, 所谓的四大支柱, 其中的传统和纪律就是为了让学生变成流水线的产品而已, 对于不符合规矩的一律打压, 当然在中国其实更为严重, 其实你我又与流水线有多大的差别呢…

饮食男女

人生不像做菜, 不能等所有的食材都准备好才下锅. 我很喜欢的一句话, 毕竟你也不知道接下来会发生什么, 人生不是上学或者上班, 带着很强的目的性. 人生更重要的在于经历着.

有关圆周率的碰撞问题

 Mar 3, 2019

  • n = 1, 发生3次碰撞
  • n = 1e2, 发生31次碰撞
  • n = 1e4, 发生314次碰撞
  • n = 1e6, 发生3141次碰撞
  • n = 1e8, 发生31415次碰撞

随机行走

 Feb 2, 2019

随机行走在随机过程中是一个很简单而又很经典的例子, 以一维随机行走为例, 每一时刻都按照概率\(p\)向右走一格, 按照概率\(1-p\)向左走一格. 我们记\(S_n\)为第\(n\)步走到的位置, 那么 $$S_n= \begin{cases} S_{n-1}+1& \text{rand < p}\\
S_{n-1}-1& \text{others} \end{cases}$$
