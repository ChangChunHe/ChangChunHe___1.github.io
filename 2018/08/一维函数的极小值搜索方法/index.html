一维函数的极小值搜索方法

August 15, 2018 in optimization

求一个函数的极小值或者是最小值一般来说是一个很复杂的问题, 涉及到全局优化的问题, 从最开始的梯度下降, 共轭梯度, 牛顿法以及衍生出来的准牛顿法等等都可以较好地搜索到局域极小值. 这里我们先以一元函数为例, 简述求解其极小值的几种方法.

首先假设我们新建一个一维优化问题的类

import numpy as np

class OneDOptAlgo(object):

    def __init__(self, f, except_inter=1e-3):
        self.f = f # here f is a one dimensional function
        self.except_inter = except_inter

我们所谓希望寻找到函数的极小值点也即为不断地压缩解区间, 直到区间长度到达我们需要的精度为止.

1. 黄金分割法

所谓黄金分割法即为我们使用黄金分割的比例压缩区间. 例如给定区间[a, b], 我们利用黄金分割比rho = 1-0.618, 新建一个区间, 其左端点anew为a+rho*(b-a),其右端点bnew为a+(1-rho)*(b-a). 如果新区间的左端点的函数值小于右端点的函数值, 那么极小值点一定落在[a, bnew], 否则的话极小值点一定落在 [anew, b], 我们不断更新这一步骤即可不断缩小区间的长度. 画个图可以很容易看出这一结果, 这里由于本人比较懒就不画图了.

其代码如下所示

def golden_division(self, inter):
    """
    f is the given function,
    iter is the interval that variable belongs to
    """
    rho = (3 - np.sqrt(5))/2
    old_inter = abs(inter[0]- inter[1])
    while old_inter > self.except_inter:
        interval = inter[1] - inter[0]
        new_inter_0 = inter[0] + rho*interval
        new_inter_1 = inter[0] + (1 - rho)*interval
        if self.f(new_inter_0) < self.f(new_inter_1):
            inter = [inter[0], new_inter_1]
        else:
            inter = [new_inter_0, inter[1]]
        old_inter = abs(inter[0] - inter[1])
    return inter

做一个测试:

OneDOptAlgo.golden_division = golden_division
# 测试
f = lambda x :x**4 - 14*x**3 +60*x**2 - 70*x
inter = [0, 2]
ODAG = OneDOptAlgo(f)
except_interval = ODAG.golden_division(inter)
print('The initiate interval is: \n', inter,
      '\n and the width of the interval is:\n', except_interval)

实际上如果我们取rho=0.5也即为二分法, 这是一种惯用的缩小解区间的方法.

2. 斐波那契数列分割法

斐波那契数列法即为每一次压缩区间的比例不保持相同, 其比例为斐波那契数列的前一项与后一项的比值, 据说这样的比例是压缩区间期望最大的…看不懂证明过程…我只会写写代码…

def fib_division(self, inter, epsilon = 0.05):
    # calculate the Fib serial
    width = inter[1] - inter[0]
    iteration = np.ceil(np.log(width*np.sqrt(5)/self.except_inter)/np.log(1.618)-2)
    count = 0
    a, b = 1, 2
    rho = [a/b]
    for i in range(int(iteration)-1):
        a, b = b, a+b
        rho.append(a/b)
    rho = rho[::-1]

    while count < iteration:
        interval = inter[1] - inter[0]
        if count == iteration - 1:
            new_inter_0 = inter[0] + (1 - rho[count]-epsilon)*interval
            new_inter_1 = inter[0] + rho[count]*interval
        else:
            new_inter_0 = inter[0] + (1 - rho[count])*interval
            new_inter_1 = inter[0] + rho[count]*interval

        if self.f(new_inter_0) < self.f(new_inter_1):
            inter = [inter[0], new_inter_1]
        else:
            inter = [new_inter_0, inter[1]]
        count += 1
    return inter

下面讲两个通过插值方式来近似曲线进而求解函数极小值的方法.

3.割线法

我们知道, 如果我们知道一元函数上某点切线斜率的符号, 就可以知道极小值点在其左边还是右边, 但是一般而言求出切线是比较困难的, 退而求其次我们可以很容易求一条割线来近似切线, 进而求解函数的极小值点.

def secant_way(self, init_point, f):
    x0 =  init_point
    if diff_f(x0) > 0:
        x1 = x0 - 0.1
    else:
        x1 = x0 + 0.1
    x2 = x1 - (x1 - x0)/(diff_f(x1) - diff_f(x0))*diff_f(x1)
    while abs(x2 - x1) > self.except_inter:
        x0, x1 = x1, x2
        x2 = x1 - (x1 - x0)/(diff_f(x1) - diff_f(x0))*diff_f(x1)
    return x2

4. 逆抛物线插值法

更一般地, 我们也可以通过三个点来插值一条抛物线, 从而实现对原函数的二阶近似. 该方法也被称为逆抛物线插值法.因此我们需要先选定三个初始点\((x_1, f(x_1)), (x_2, f(x_2)), (x_3, f(x_3))\), 以这三个点确定一条抛物线, 然后求该抛物线的极值点即为

$$x_{new} = \frac{(x_1^{2}-x_2^{2})f(x_3)+(x_2^{2}-x_3^{2})f(x_1)+(x_3^{2}-x_1^{2})f(x_2)}{2f(x_3)(x_1-x_2)+2f(x_1)(x_2-x_3)+2f(x_2)(x_3-x_1)}$$

以此类推的话, 我们可以给出更高阶的递推公式.

def reverse_parabola(self, init_point):
    f = self.f
    x3 = init_point
    x1, x2 = init_point + 0.1, init_point - 0.1
    err = self.except_inter*2
    while err > self.except_inter:
        sig12, sig23, sig31 = x1**2 - x2**2, x2**2 - x3**2, x3**2 - x1**2
        delt12, delt23, delt31 = x1-x2, x2-x3, x3-x1
        xnew=
        (sig12*f(x3)+sig23*f(x1)+sig31*f(x2))/
        (2*(delt12*f(x3)+delt23*f(x1)+delt31*f(x2)))
        x3, x2,x1 = xnew, x3, x2
        err = abs(x3-x2)
    return x3
