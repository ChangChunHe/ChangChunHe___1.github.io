MATLAB如何读取和写入文本文件

August 12, 2018 in Coding

这篇博客来讲一讲MATLAB如何读取文本文件.

如果是标准的矩阵类型的文本文件, 可以使用data = load(filename)来快速便捷读取文件并储存在变量data中, 这里就不再赘述.下面主要讲讲几种较为复杂的文本文件的读取, 以及简单的文件写入.

1. 读取文件

在matlab中, 我们使用fid = fopen(filename, 'r')来读取一个文件(r表示read的含义), 并返回文件的id.

  • fgetl

使用tline = fgetl(fid)可以返回文件的第一行, 当然是字符类型的数据. 再次调用这个语句就可以返回第二行的数据(如果有的话).所以加入文件有10行, 我们只需要重复调用10次该语句就可以返回文件的所有内容. 但是一般我们并不知道文件有多少行, matlab当然考虑到这一点, 我们使用feof(fid)来确认是否是最后一行, 否则就退出读取.

fid = fopen(filename, 'rt');
while feof(fid) == 0
    tline = fgetl(fid);
    fprintf('This line is %s', tline)
    %do-something-you-need
end
fclose(fid);%最后记得关闭文件

例如我们希望提取DOSCAR中的DOS, 由于是提取的DOS是标准的三列数字, 这个时候可以使用强大的str2num函数, 直接将字符串变成数字类型.就以上述的文件为例, 需要提取的数据有19行, 从7行开始, 那么我们就可以使用下面的的代码:

clear
fid = fopen('DOSCAR', 'r');
DOS = zeros(19, 3);
k = 1;
while feof(fid) == 0
    tline = fgetl(fid);
    if k >= 7
       DOS(k-6,:) = str2num(tline);
    end
    k = k + 1;
end
fclose(fid);%最后记得关闭文件
% if you want plot a figure of DOSCAR， you can use
plot(DOS(:,1), DOS(:,22), 'r')
xlabel('Energy')
ylabel('DOS')

这是一个最一般的读取文件的方式, 可以对文件进行任何细节的操作, 但是一般速度比较慢, 所以如果文件有一定的格式, 我们可以使用textscan来读取文件.

  • textscan

比如文件的前两行是字符, 我们需要读取的是第3-20行的格式化的矩阵, 不妨我们假设这个矩阵的唯独是18*20的, 那么就可以使用:

fid = fopen(filename, 'rt');
FormatString = repmat('%f ', 1, 20);
C = cell2mat(textscan(fid, FormatString, 18, 'HeaderLines', 2));
fclose(fid);

其中, FormatString表示格式, %f表示浮点数, HeaderLines表示跳过题头的若干行. 例如上述使用fgetl读取的文件, 我们可以使用 DOS = cell2mat(textscan(fid, '%f %f %f', 19, 'HeaderLines', 6))一句代码就可以读取了.

下面简要介绍格式化输出的几个常用参数

字符 含义
%d 或 %i 以 10 为基数
%m.nf 定点记数法(使用精度操作符指定小数点后的位数, m表示字符的长度的下限, n表示小数点后几位, 默认六位)
%e 指数记数法，例如 3.141593e+00(使用精度操作符指定小数点后的位数)
%g 更紧凑的 %e 或 %f，不带尾随零(使用精度操作符指定有效数字位数)
%s 字符向量或字符串数组

2.写入文件

类似的, 我们使用fid = fopen(newfilename, 'w')来新建一个文件, 接着可以对其中写入内容

  • fprintf

例如我们生成了一随机数矩阵M = rand(4), 我们期望写到文件random.txt中, 那么我们使用下面的代码:

fid = fopen('random.txt', 'w');
M = rand(4)
for ii = 1:4
    fprintf(fid, '%.5f %.5f %.5f %.5f', [num2str(M(ii,:)), '\n']);
    % 这里我们就保存了小数点后五位有效数字
    % 注意写入换行符
end
fclose(fid);

例如我们需要写入字符文件:

clear
clc
a = {'A A A', 'A B C', 'C C C'}
fid = fopen('test.txt', 'w');
for ii = 1:length(a)
    fprintf(fid, [a{ii}, '\n'])
end
fclose(fid);
